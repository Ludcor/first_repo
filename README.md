# Mi Primer Repositorio de Pruebas 🚀

¡Hola! Este es un repositorio personal que creé con el fin de aprender a usar **Git** y **GitHub**, y para realizar mis primeras pruebas de código en Python.

## 📌 Objetivos del Proyecto
- Comprender el flujo de trabajo de Git (add, commit, push).
- Aprender a solucionar errores en la terminal.
- Guardar mis scripts de práctica, como `QuizProject.py`.

## 🛠️ Comandos de Git Aprendidos
Hasta ahora he dominado los siguientes comandos en Windows CMD:
* `git commit -m "mensaje"` -> Guarda los cambios locales (¡usar comillas dobles en CMD!).
* `git push -u origin main` -> Sube los cambios a GitHub por primera vez.
